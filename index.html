How It Works

The web is a bunch of computers connected by a network. When you type in a web address known as HTTP it travels across the internet to a server which returns the information requested and your computers browser interprets the information into what you see on the computer screen.

HTML Make Up

HTML stands for Hyper Text Mark Up Language. The Text Content is what you see and the Mark Up is what is looks like. It also contains References to other documents and Links to other web page such as GOOGLE

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
